# Poker Chart

App web installabile su iPhone per visualizzare chart preflop poker in base a:

- numero giocatori al tavolo: 5, 6, 8, 9
- posizione
- stack da 10 a 100 BB

## Pubblicazione con GitHub Pages

1. Crea un nuovo repository su GitHub.
2. Carica tutti i file di questa cartella nel repository.
3. Vai in Settings -> Pages.
4. In "Build and deployment", seleziona "Deploy from a branch".
5. Seleziona branch `main` e cartella `/root`.
6. Salva.

Dopo qualche minuto GitHub pubblichera l'app a un indirizzo simile:

`https://TUO-USERNAME.github.io/NOME-REPOSITORY/`

Da iPhone apri quell'indirizzo con Safari, poi usa Condividi -> Aggiungi alla schermata Home.
